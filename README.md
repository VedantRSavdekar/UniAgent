# 🤖 UniAgent - AI Job Search Assistant

An AI-powered job search assistant built with Python, Streamlit, and LangChain.

## 🔗 Live Demo
[Click here to try UniAgent](https://uniagent-dk6ilm9cmfascg5mtppwlm.streamlit.app/)

## ✨ Features
- 📄 **Resume Parser** — Upload your resume (PDF/DOCX) and extract key information
- 💼 **Job Matcher** — Paste a job description and get an AI match analysis
- 💬 **Chat** — Ask questions about your resume and get career advice
- 🎯 **Interview Prep** — Generate interview questions and practice with AI feedback
- 💾 **History** — View all past resumes and job matches

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- Groq (LLaMA 3.1)
- Google Gemini
- SQLite

## 🚀 Run Locally

1. Clone the repo
```bash
git clone https://github.com/VedantRSavdekar/UniAgent.git
cd UniAgent
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Create a `.env` file
```bash
# API keys format
GROQ_API_KEY=your_groq_key
GEMINI_API_KEY=your_gemini_key
```

4. Run the app
```bash
streamlit run app.py
```
